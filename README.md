# Website Portofolio Pribadi

Website portofolio ini dibuat sebagai media untuk memperkenalkan diri, menampilkan pengalaman, keterampilan, serta proyek yang pernah dikerjakan. Website ini dibangun menggunakan Tailwind CSS dengan fitur mode terang (light mode) dan mode gelap (dark mode) yang memiliki skema warna berbeda, serta dirancang responsif untuk berbagai ukuran layar.

---

## Deskripsi

Website ini bertujuan untuk memberikan gambaran mengenai latar belakang, pengalaman, dan kemampuan saya di bidang teknologi informasi, khususnya web development dan IT support. Dengan tampilan yang sederhana, modern, dan mudah digunakan, website ini diharapkan dapat menjadi sarana dokumentasi sekaligus representasi profesional.

---

## Fitur

- Mode terang dan mode gelap dengan skema warna yang berbeda
- Desain responsif untuk desktop, tablet, dan perangkat mobile
- Navigasi sederhana dan mudah digunakan
- Tampilan modern dengan animasi dan transisi ringan

---

## Struktur Konten

Website portofolio ini terdiri dari beberapa bagian utama sebagai berikut:

### Pengenalan Diri
Berisi informasi singkat mengenai profil, minat, dan latar belakang saya di bidang teknologi dan sistem informasi.

### Pengalaman Kerja dan Pendidikan
Menampilkan riwayat pengalaman kerja, magang, serta pendidikan yang relevan dengan bidang teknologi informasi.

### Keterampilan
Berisi daftar keterampilan dan tools yang sering digunakan, antara lain HTML, CSS, JavaScript, Tailwind CSS, serta tools desain dan pendukung lainnya.

### Proyek
Menampilkan beberapa proyek yang pernah dikerjakan, dilengkapi dengan deskripsi singkat dan tautan demo.

### Galeri
Berisi kumpulan dokumentasi dan serpihan momen yang terekam selama beberapa tahun terakhir sebagai bagian dari perjalanan dan pengalaman.

### Kontak
Menyediakan informasi kontak dan form untuk keperluan komunikasi, diskusi, maupun kolaborasi.
